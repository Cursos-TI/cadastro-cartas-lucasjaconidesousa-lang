# Super Trunfo em C

Projeto desenvolvido em linguagem C para o desafio nível mestre.

## Funcionalidades:
- Cadastro de duas cartas
- Cálculo de densidade populacional
- Cálculo de PIB per capita
- Cálculo do Super Poder
- Comparação entre cartas sem uso de if/else
